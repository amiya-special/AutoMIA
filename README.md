# AutoMia

AutoMia 相关代码与 Agent 实现，用于 VL-MIA（Vision-Language Membership Inference Attack）实验。

## 快速开始

- **详细运行说明**请见：[llava/README.md](llava/README.md)
- 数据集：VL-MIA [image](https://huggingface.co/datasets/JaineLi/VL-MIA-image) / [text](https://huggingface.co/datasets/JaineLi/VL-MIA-text)

## 目录结构

- `agent_mia/` — Agent 与配置
- `llava/` — LLaVA 相关脚本与运行说明（`run_with_text_agent.py`、`run_with_img_agent.py`）
